# Candidate Application — Business Specification

### Overview
This feature allows interested individuals (candidates) to apply for published job openings. It provides a public-facing application form where candidates can submit their personal details and professional resume. This is the primary intake mechanism for the ATS, replacing manual email submissions with a structured, validated process that prevents duplicate applications for the same role.

### User Stories
- As a **guest visitor**, I want to view the details of a job posting, so that I can decide if I am qualified to apply.
- As a **candidate**, I want to fill out a simple application form with my name, email, and phone number, so that the company has my contact information.
- As a **candidate**, I want to upload my resume as a PDF file, so that the recruiter can review my professional background.
- As a **candidate**, I want to receive an immediate confirmation after submitting my application, so that I know it was successfully received.
- As a **company administrator**, I want the system to prevent a candidate from applying to the same job multiple times using the same email address, so that I don't have to manage redundant applications.
- As a **company administrator**, I want all submitted applications to automatically appear in the "Applied" stage of my candidate tracking dashboard, so that I can begin the evaluation process immediately.
