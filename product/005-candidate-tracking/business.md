# Candidate Tracking — Business Specification

### Overview
This feature provides the central workspace for recruiters and administrators to manage the pipeline of candidates. It enables manual progression of applicants through defined hiring stages, allows for structured evaluation via notes and scores, and ensures that every application reaches a definitive terminal state (Archive) with a recorded resolution.

### User Stories
- As a **company administrator**, I want to see a clear overview of all candidates grouped by their current hiring stage, so that I can prioritize my daily recruitment activities.
- As a **company administrator**, I want to search for candidates by name or email, so that I can quickly find a specific applicant.
- As a **company administrator**, I want to filter candidates by a specific job posting, so that I can review the talent pool for a particular opening.
- As a **company administrator**, I want to move a candidate from one stage to another (e.g., from Applied to Interview), so that the system reflects their current progress in the hiring process.
- As a **company administrator**, I want to record evaluation notes and a numerical score (1 to 5) for a candidate, so that I can document my assessment for future reference.
- As a **company administrator**, I want to archive a candidate's application by selecting a specific resolution (Hired, Rejected, Declined, or Withdrawn), so that the hiring cycle is properly concluded and tracked.
- As a **company administrator**, I want to view the full history of feedback and stage changes for a candidate, so that I have complete context on their journey.
