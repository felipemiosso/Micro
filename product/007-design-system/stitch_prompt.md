# Stitch Enhanced Prompt: Honeycomb ATS Design System

**Title:** Professional ATS Interface - Honeycomb Theme
**Filename:** honeycomb_ats_design_system.md

## High-Level Intent
Create a "Clean and Professional" design system for an Applicant Tracking System (ATS). The aesthetic should be "Sharp and Sophisticated," using a Honeycomb-inspired palette (Ambers, Golds, and Deep Charcoals). The UI must feel data-dense yet readable, suitable for HR professionals managing large applications board.

## Theme & Vibe
- **Colors:** Deep Charcoal (#111827) for text and primary structure; Rich Amber (#F59E0B) and Gold (#FBBF24) for primary actions and highlights. Use a very light Amber-tinted Gray (#F9FAFB) for page backgrounds.
- **Typography:** Use **Inter** (or a clean system sans-serif) with tight letter spacing for a modern, professional feel.
- **Styling:** 8px rounded corners (rounded-lg), subtle 1px borders, and soft shadows for depth. Incorporate subtle hexagonal patterns or motifs in backgrounds or icon sets.

## Component Specifications
- **Primary Button:** Solid Amber background, white text, bold font, subtle lift on hover.
- **Secondary Button:** Ghost style with Amber border and text, transitions to light Amber background on hover.
- **Cards:** White background, 1px Gray-200 border, slight shadow. Use a vertical Amber "accent stripe" on the left edge for active/selected states.
- **Applications Board Lanes:** Subtle tinted backgrounds (Gray-50) with high-contrast headers.
- **Status Badges:** Use a "Pill" shape with low-opacity Amber backgrounds and high-saturation text (e.g., bg-amber-100 text-amber-800).

## Specific Page Guidelines (Dashboard/Applications Board)
- **Board View:** Ensure cards have clear hierarchy (Candidate Name, Role, Rating, Status).
- **Navigation:** Deep Charcoal side-nav or top-nav with gold active-state indicators.
